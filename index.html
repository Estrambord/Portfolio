This is a school project where I had to choose one of the many Star Wars droid, 3D model it and create a whole scene around it. This project taught me how to use modelling software, texture software and Unity. I had never used any of the softwares before so I'm quite proud of how it turned out in the end.

Kinect Sense Game

This Kinect game for 3 players is a good way to put your communication skills to the test ! All 3 players start the game with a sense removed : one is blindfolded, one is made deaf by wearing noise-cancelling headphones, and the last one cannot talk. And yet they will have to play as a team through visual and auditive games.

A fun and challenging experience with a simple Kinect device that we developped during 2 weeks with a team of 5 people.

View Project

Securing a Flowforming Machine

On my campus at Arts et Métiers in Cluny there was an old Flowforming Machine that was barely used because it was too old to respect the current safety standards. With a team of 3 classmates, our project was to dismantle the old electronic circuits, study them (because all the documentation had disappeared) and rebuild a working and safe electronic system.
