Star Wars Droid Project

Project I conducted alone, during a few months to learn how to do a realtime 3D scene from start to finish.

About the project

This is a school project that was made to teach us the making of a realtime 3D scene from start to finish. I had never done any 3D modelling nor texture before so it was tough to learn how to use so many softwares at once but very interesting at the same time. Over these two months I spent working on my scene, I quickly learned a lot of skills and overall I'm very proud of the result for a beginner.

The theme of the project was : choose a Star Wars droid, model it and a scene to go around it, and then set up some lighting to create a beautiful realtime 3D scene. I chose a simple droid, JV-P12 from Solo: A Star Wars Story to learn 3D modelling and increased the difficulty by making more complex props for my scene.

I am still adding a few details to this project, a video of the realtime 3D scene is coming soon and will be added to the gallery, showing in depth lighting and details in the scenery.

This scene is in real time using Unity HDRP pipeline and is suitable for real-time 3D applications and virtual reality.
